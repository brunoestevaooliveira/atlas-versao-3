# Como adicionar a imagem no GitHub

Para exibir a imagem do Atlas Cívico no README, siga estes passos:

## Opção 1: Upload direto no GitHub (Recomendado)

1. **Acesse o repositório**: https://github.com/brunoestevaooliveira/atlas-versao-3
2. **Crie uma pasta assets**:
   - Clique em "Create new file"
   - Digite: `assets/images/screenshot-main.png`
   - Isso criará a pasta automaticamente
3. **Faça upload da imagem**:
   - Arraste a imagem da interface do Atlas Cívico
   - Commit com mensagem: "Adicionar screenshot da interface principal"

4. **Atualize o README**:
   - Edite o README.md
   - Substitua o link da imagem por:
   ```markdown
   ![Atlas Cívico v3 - Interface Principal](./assets/images/screenshot-main.png)
   ```

## Opção 2: Usando Issues/Discussions (Mais fácil)

1. **Abra uma nova Issue** no GitHub
2. **Arraste a imagem** diretamente no campo de descrição
3. **Copie o link gerado** (será algo como: `https://github.com/user/repo/assets/123456/...`)
4. **Use esse link no README**

## Resultado esperado

Depois de adicionar, a imagem aparecerá no README mostrando:
- ✅ Mapa interativo de Santa Maria-DF
- ✅ Painel de ocorrências recentes
- ✅ Interface moderna e responsiva
- ✅ Sistema de filtros e busca

Isso deixará o README muito mais atrativo e profissional! 🎨
